# Quick 260721-i66: Deterministic Alpha-Correct Bilinear Summary

Added a public, checked bilinear resize that filters supported sRGB RGB8 and straight-RGBA8 input in linear premultiplied space, preventing transparent-colour fringes while retaining deterministic byte vectors.

## Delivered

- Added `@ops.resize_bilinear` with nearest-compatible checked coordinate bases, high-neighbour clamping, four-tap work charging, and one authoritative output allocation.
- Added shared package-visible RGB8 and straight-RGBA8 linear-premultiplied load/store helpers; zero-alpha RGBA stores as black.
- Added exact public and white-box coverage for linear-light interpolation, transparent-red halo removal, coordinate fractions, typed rejection, checked overflow, and complete budget atomicity.
- Published executable public vectors and the portable target command matrix in the mb-image README.
