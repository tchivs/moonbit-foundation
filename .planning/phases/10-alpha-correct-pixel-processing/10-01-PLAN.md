## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| caller ImageView -> processing operations | Caller-controlled representation, dimensions, pixels, and metadata enter portable pixel loops. |
| caller metadata -> composite output descriptor | Profile, orientation, and opaque metadata from two inputs must be reconciled before a result can be described. |
| caller Budget -> owned output allocation | Resource limits constrain byte storage, allocation count, dimensions, pixels, and declared work. |

| Threat ID | Category | Component | Severity | Disposition | Mitigation Plan |
|-----------|----------|-----------|----------|-------------|-----------------|
| T-10-01 | Tampering | processing capability gate | medium | mitigate | Require straight packed U8 RGBA with sRGB/encoded-sRGB metadata and return operation-specific CapabilityUnavailable before allocation. |
| T-10-02 | Denial of service | box_blur radius/window/work preflight | high | mitigate | Checked add/multiply computes window and total sample work before the single owned allocation; supplied Budget atomically rejects insufficient limits. |
