<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| caller Reader → transport/DEFLATE | Untrusted PNG framing and compressed bytes control chunk, bit, tree, and match parsing. |
| DEFLATE sink → raster/image | Untrusted decompressed bytes select filters and attempt output expansion. |
| local OwnedImage → DecodeResult | Allocated pixels must remain private until every checksum and terminal PNG condition passes. |

## STRIDE Threat Register

| Threat ID | Category | Component | Severity | Disposition | Mitigation Plan |
|-----------|----------|-----------|----------|-------------|-----------------|
| T-21-01 | Tampering | IDAT transport | high | mitigate | CRC each physical chunk, preserve order/policy checks, and consume one logical stream. |
| T-21-02 | Denial of service | DEFLATE/raster | high | mitigate | Fixed 32 KiB history, bounded tree alphabets, exact sink count, existing limits, and authoritative OwnedImage budget charge. |
| T-21-03 | Tampering | zlib/DEFLATE | high | mitigate | Validate wrapper, stored lengths, canonical trees, symbols, distances, Adler-32, and trailing compressed bytes. |
| T-21-04 | Elevation | eager result | medium | mitigate | Construct DecodeResult only after inflate, raster count, IDAT tail, IEND, and EOF succeed. |
</threat_model>
