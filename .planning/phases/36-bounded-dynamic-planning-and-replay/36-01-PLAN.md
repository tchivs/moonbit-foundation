<interfaces>
Existing contracts to preserve and extend:

- `PngCompressionStrategy::{Stored, FixedOrStored, DynamicOrFixedOrStored}` is already public; only the third route may select Dynamic.
- `_png_encode_preflight(source, strategy, limits, budget)` is the one shared eager/chunk admission and budget-charge path. Its existing `PngDeflatePlan::{Stored, Fixed}` feeds `PngEncodeMachine::new_with_compression_strategy`.
- `_png_huffman_new(lengths, require_eob, allow_single_distance)` validates canonical lengths and returns codes already reversed for DEFLATE's least-significant-bit stream. `_png_dynamic_trees` in `deflate_inflate.mbt` defines the decoder's required HLIT/HDIST/HCLEN permutation and RLE semantics.
- `PngEncodeMachine::present()` caches one pending byte; `acknowledge(byte)` is the sole transition that changes emitted count, CRC, Adler, and the plan-specific preview state. `PngChunkEncoder` already latches the first machine failure and scopes output leases to a pull.
