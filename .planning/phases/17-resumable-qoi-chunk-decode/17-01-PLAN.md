## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| caller ByteView → QoiStreamDecoder.push | Untrusted temporary input drives parser progress; the decoder must not retain caller memory. |
| parsed QOI header → owned-image allocation | Untrusted dimensions, channels, and declared work can request resource consumption. |
| parser state → callback-scoped mutable image view | Long-lived parser data crosses multiple pumps while mutable storage authority is intentionally short-lived. |
| compiler interface/source inventory → QOI quality policy | Generated public API and package contents are checked against the candidate policy allowlist. |
