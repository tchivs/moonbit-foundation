## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Caller metadata → descriptor | Caller-provided format, plane, alpha, transfer, profile, and orientation identities determine whether a packed image is valid. |
| Descriptor → storage/view | A validated descriptor determines allocation length, channel bounds, and byte offsets for caller-visible pixel data. |
| Image view → operations | An image model that is storable is not automatically authorized for color-processing or copy semantics. |
