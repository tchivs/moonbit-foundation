## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Public eager PNG bytes -> public decoder | Generated codec output crosses a complete-input parser boundary before its descriptor and samples become the fidelity oracle. |
| Public chunk encoder -> caller-owned byte lease | Each pull exposes only bytes accepted into a capacity-limited caller buffer and reports cumulative progress. |
