## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| GitHub environment to publisher process | The only production credential crosses into one isolated step. |
| Prepared artifact/journal to mutation adapter | Untrusted cross-job data decides which irreversible action, if any, is eligible. |
| Mutation outcome to observer/consumer | Publish process output is not success evidence; public exact observation and cold proof are separate. |
