## Public finish() classification (frozen for implementation)

The adapter must use one private, non-mutating classifier over active `PngDecodeMachineState`; it must not invoke `PngDecodeMachine::finish()` except in `NeedEof`, and must store its resulting error before returning. Every table result is `Data/UnexpectedEndOfStream` with `completed` equal to the caller-visible consumed total, unless the table says to reuse the existing typed machine error.

| Priority | Active condition at finish() | Required terminal result | Required test coverage |
|----------|------------------------------|--------------------------|------------------------|
| 1 | wrapper already Failed(error) | replay exactly that original CoreError; no classifier replacement | malformed signature/CRC, limit, allocation/budget, trailing, and prior zlib errors followed by finish/push |
| 2 | an IDAT inflater exists but zlib header, DEFLATE token/tree/match, or Adler trailer is incomplete | existing typed `zlib-truncated` | partial stored/fixed/dynamic/Adler input while Length, Type, Payload, or Crc framing is also incomplete |
| 3 | inflater is complete but the private raster sink is incomplete | existing typed `png-filtered-size` | complete zlib stream with a short filtered row/Adam7 pass before a next header, partial IEND, and missing IEND |
| 4 | Signature(index) | `png-signature` | every signature prefix length |
| 4 | Length(bytes) before completed IDAT/raster | `png-chunk-length` | one through three generic length bytes |
| 4 | Length([]) after completed inflater and raster | `png-iend` | stream ending immediately after the last authenticated IDAT CRC |
| 4 | Length(one to three bytes) after completed inflater and raster | `png-iend-length` | each partial IEND length boundary |
| 4 | Type(length, kind) after completed inflater and raster | `png-iend-length` when declared length is nonzero; otherwise `png-iend-type` for every partial or mismatching IEND type prefix | zero/nonzero declared length plus I, IE, IEN, wrong fourth type, and completed non-IEND type |
| 4 | Payload(…, IEND, …) | unreachable by construction: fourth IEND type byte rejects a nonzero length as existing `png-iend-length`; retain this invariant and test the public attempted-payload route | nonzero-IEND declaration fails before a payload byte can be accepted |
| 4 | Crc(IDAT, …) after priorities 1-3 | `png-idat-crc` | zero through three CRC bytes after a complete IDAT payload |
| 4 | Crc(IEND, …) after priorities 1-3 | `png-iend-crc` | zero through three IEND CRC bytes; explicit IEND CRC context is mandatory |
| 4 | other Payload/Crc | `png-chunk-payload` / `png-chunk-crc` | partial ancillary and ordinary chunk payload/CRC cases |
| 5 | NeedEof | call private `PngDecodeMachine::finish`; success is the only transfer point | full valid IEND + finish success, plus private terminal integrity/raster errors |

This priority applies before all framing suffixes: incomplete Adler/DEFLATE wins over IDAT/IEND truncation, and incomplete raster wins over missing/partial IEND once zlib is complete. A first machine error remains sticky even if a later call would otherwise match a higher table row.

## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Caller-owned ByteView → PngChunkDecoder::push | Untrusted mutable bytes cross a synchronous call boundary; the decoder must not retain a caller view. |
| PngChunkDecoder → private PngDecodeMachine | Public consumption/terminal state must preserve the verified parser's integrity, resource, and error semantics. |
| private PngMachineOutcome → DecodeResult | The sole allocated image is transferred only after explicit EOF and all final integrity checks. |
