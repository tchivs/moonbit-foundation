## Cross-Phase Integration

- Phase 47 established component-byte access plus a Gray16 PNG profile. `_png_wire_byte` converts host-endian storage into PNG's big-endian wire order, and public eager/chunked `new_gray16` factories select the profile.
- Phase 48 verifies the profile through compression planning, filtering, replay, atomic preflight, and the Gray16 Adam7 guard, including non-symmetric U16 values that expose byte-order mistakes.
- Phase 49 exercises only the public factories for eager and chunked encoders: exact stored scanline payloads, canonical decode, incremental draining, sticky terminal state, and progress bounds.
- Frozen Gray8, RGB, and RGBA vectors remain stable. The PNG package passed `190/190` tests on each declared target: JS, Wasm, Wasm-GC, and native.
