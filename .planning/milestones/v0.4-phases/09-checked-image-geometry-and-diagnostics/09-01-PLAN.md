<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| caller ImageView and Rect to geometry operation | Untrusted dimensions, region endpoints, pixel layout, and budget enter portable allocation and write logic. |
| validated descriptor to OwnedImage allocation | Checked geometry becomes a single storage, allocation, dimension, pixel, and work budget charge. |

## STRIDE Threat Register

| Threat ID | Category | Component | Severity | Disposition | Mitigation Plan |
|-----------|----------|-----------|----------|-------------|-----------------|
| T-09-01 | Tampering | crop region validation in geometry.mbt | high | mitigate | Validate Rect endpoints and source bounds before descriptor construction and use checked multiplication for every output scalar. |
| T-09-02 | Denial of service | OwnedImage output allocation | high | mitigate | Pass only a validated tight descriptor and explicit work to OwnedImage::new_operation, whose atomic budget charge occurs before mutable writes. |
| T-09-03 | Tampering | pixel format capability gate | medium | mitigate | Reuse supports_copy_flip so only packed U8 encoded-sRGB RGB/RGBA formats reach byte-level traversal. |
| T-09-04 | Tampering | explicit rotation metadata | medium | mitigate | Normalize physical rotation output to TopLeft and report orientation as the sole transformed metadata field. |
</threat_model>
