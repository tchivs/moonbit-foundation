siloed private state machine: the public `PngChunkDecoder` owns and calls the

## End-to-End Flows

1. **Public resumable decode:** caller-owned `ByteView` → `PngChunkDecoder.push` → private machine byte acceptance → explicit `finish` → one `DecodeResult` or sticky error.  Corpus evidence covers empty, one-byte, and ragged schedules for all 3,850 generated accepted/rejected records.
2. **Portable image processing:** sixteen public pushes for the 75-byte input → `finish` → bilinear resize → eager `PngEncoder` → 78-byte output, digest `626208771`.
